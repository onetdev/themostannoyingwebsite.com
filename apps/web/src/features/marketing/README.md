# Promotion feature

Displays intrusive advertisements, promotional content and fake rewards.

## Key highlights

- **DilfPage** and **FlaimSurvey** components for App Router integration.
- **DILF promotion** on both sides of the content leading to donut finder
- **Fake quiz with animated promotion banner**.
- **Detecting adblocker** presence and warning user through a banner at the bottom of the page
- **Fake wheel of fortune** that provides no real rewards.
- **Newsletter dialog** that pops up:
  - Once the user scrolls down enough
  - On tab switches to another tab and then coming back

# Out of scope

- Working newsletter subscription
- Working backend integration
- Any real rewards
