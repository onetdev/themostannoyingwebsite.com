このポリシーは、当社がお客様から収集する情報、その情報の使用方法、およびその法的根拠について説明するものです。また、情報の共有方法、および当社に提供された情報に関するお客様の権利と選択肢についても説明します。このプライバシーポリシーは、お客様による mostannoyingwebsite.com (The MAW) ウェブサイトの利用を通じて当社が取得する情報に適用されます。

### 当社について

The MAW は、当社の <a href={config.githubUrl}>Git リポジトリ</a> を通じて公開されているオープンソースプロジェクトです。このプライバシーポリシーに関するご質問は、<a href={`mailto:${config.contactEmail}`}>{config.contactEmail}</a> までお問い合わせください。当社がお客様の個人データを第三者に販売することはありません。

### 収集する情報

当社のウェブサイトで最高の体験を提供するために、当社は特定の情報を収集し、処理する必要があります。サービスの利用状況に応じて、これらには以下が含まれる場合があります。

- **利用データ** — お客様が当社のウェブサイトを訪問した際、当社は以下の情報を保存します。参照元ウェブサイト、訪問したページ、訪問日時と滞在時間、匿名化された IP アドレス、および訪問中に使用されたデバイスに関する情報（デバイスタイプ、OS、画面解像度、言語、国、ブラウザタイプ）。当社は、統計目的、ウェブサイトの改善、および不正利用の特定と防止のために、Vercel Analytics でこれらの利用データを処理します。
- **エラー報告** — 当社のウェブサイトでは、信頼性向上のために Sentry を使用してエラー報告を収集・監視しています。このデータには、匿名化された IP アドレス、エラーメッセージ、デバイス情報、およびブラウザの詳細が含まれる場合があります。詳細については、Sentry のプライバシーポリシー（https://sentry.io/privacy/）を参照してください。
- **クッキー** — 当社は、記録保持およびウェブサイトの機能向上のためにクッキー（ウェブサイトからコンピュータまたはデバイスに転送される小さなデータファイル）を使用します。ブラウザの設定を調整することで、クッキーの転送を無効化または制限できます。すでに保存されているクッキーは、いつでも削除できます。
- **埋め込み YouTube 動画** — 当社のウェブサイトには、Google Inc. が運営する YouTube の埋め込み動画が含まれる場合があります。埋め込み YouTube 動画を含むページを閲覧すると、YouTube は IP アドレス、デバイス情報、ブラウジングの詳細など、動画とのやり取りに関する情報を収集・処理する場合があります。詳細については、Google のプライバシーポリシー（https://policies.google.com/privacy）を参照してください。
